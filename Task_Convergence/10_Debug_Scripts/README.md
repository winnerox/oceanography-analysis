# 10_Debug_Scripts - 数据检查/调试脚本

## 功能说明

本文件夹包含用于检查数据结构、调试代码问题的辅助脚本。

## 文件列表

| 文件名 | 功能 |
|--------|------|
| `Inspect_IAP_NC.m` | 检查 IAP NC 文件结构和变量名 |
| `Inspect_Ishii_NC.m` | 检查 Ishii NC 文件结构和变量名 |
| `Inspect_Ishii_Mean.m` | 检查 Ishii 平均态数据内容 |
| `Check_IAP_Ishii_Vars.m` | 检查 IAP/Ishii 变量名一致性 |
| `Check_StdRef_Content.m` | 检查标准参考态数据内容 |
| `Check_Terms_Content.m` | 检查泰勒展开项数据内容 |
| `Test_GSW_Ishii.m` | 测试 GSW 库在 Ishii 数据上的兼容性 |

## 使用场景

### 数据结构检查

当遇到 NC 文件读取错误时，使用 `Inspect_*.m` 脚本检查：

- 文件是否存在
- 变量名是否正确 (不同数据集可能使用不同命名)
- 维度是否匹配

### GSW 兼容性测试

当 GSW 函数报错时，使用 `Test_GSW_Ishii.m` 测试：

- 经度格式是否正确 (-180~180 vs 0~360)
- 纬度范围是否在有效范围内
- 压力计算是否正常

### 数据内容验证

使用 `Check_*.m` 脚本验证：

- 数据是否包含预期变量
- 数值范围是否合理
- 是否存在异常值

## 常见问题排查

### 问题1: 找不到变量

```
错误: 变量 'temp' 不存在
```

**解决**: 运行 `Inspect_*_NC.m` 查看实际变量名，可能是 'temperature' 或 'pt'

### 问题2: GSW 计算失败

```
错误: gsw_SA_from_SP 计算失败
```

**解决**: 运行 `Test_GSW_Ishii.m` 检查经纬度格式

### 问题3: 维度不匹配

```
错误: 矩阵维度必须一致
```

**解决**: 运行 `Check_*_Content.m` 检查各变量维度

## 注意事项

- 这些脚本仅用于调试，不是分析流程的一部分
- 运行前请确保数据路径正确
- 可根据需要修改脚本中的数据路径
