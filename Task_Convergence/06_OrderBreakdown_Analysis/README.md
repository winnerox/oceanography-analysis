# 06_OrderBreakdown_Analysis - 阶数分解分析

## 功能说明

本文件夹包含用于分析不同阶数泰勒展开项大小和空间分布的脚本。

## 文件列表

| 文件名 | 参考态 | 阶数范围 | 功能 |
|--------|--------|----------|------|
| `Analysis_OrderBreakdown_Avg.m` | 平均态 | 1-3阶 | 分析各阶项的贡献 |
| `Analysis_OrderBreakdown_Std.m` | 标准态 | 1-8阶 | 分析高阶项的衰减规律 |

## 分析内容

### 阶数贡献分解

将 TSLA 分解为各阶项的贡献：

- **1阶项**: 线性响应 (最大贡献)
- **2阶项**: 非线性修正
- **3阶项**: 高阶修正
- **...更高阶**: 逐渐衰减

### 输出图表

1. 各阶项的空间分布图
2. 各阶项的统计分布箱线图
3. 阶数贡献占比饼图

## 科学意义

- 量化各阶项的相对重要性
- 验证高阶项的快速衰减假设
- 为选择合适的截断阶数提供依据

## 运行顺序

1. 确保已完成 `03_Terms_Calculation`
2. 运行对应参考态的分析脚本

## 依赖

- 泰勒展开项数据: `*_Terms_*.mat`
- m_map 工具箱 (地图投影)
